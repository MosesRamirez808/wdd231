Site Plan: My Bucket List

Site Name

My Bucket List

The site name was chosen because it clearly communicates the purpose of the website: a personal collection of travel destinations I hopes to visit. The name is simple, memorable, and relatable, making it easy for users to understand the site's intent at a glance.

Optional domain: mybucketlist.travel

Site Purpose

The purpose of this website is to showcase a personal travel bucket list of destinations from around the world. The site provides visitors with brief descriptions of each location, reasons for visiting, and highlights of what makes each place unique. Users can explore featured destinations or browse all locations by continent.

Scenarios

  • What destinations are included on this travel bucket list?
  • Can I filter destinations by continent to find places in a specific region?
  • Which destinations are considered “must visit” locations?

Color Scheme

#0A2540 (Navy Blue) - Used for the header, footer, navigation, and main headings.

#F4F8FC (Light Blue) - Used as the main background color for the site.

#0B3C5D (Accent Blue) - Used for buttons, badges, and interactive elements.

Typography

  • Primary Font: System UI / Segoe UI / Roboto / Arial - Used for body text and paragraphs.
  • Headings: Same font family with increased size and weight for consistency and readability.

A system font stack was chosen to ensure fast loading, high readability, and consistent appearance across different devices and operating systems.
